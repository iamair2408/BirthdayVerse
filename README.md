# ✨ BirthdayVerse

## 🌟 Key Features

- 🎨 **Glassmorphism Design:** Soft blur gradients, rounded card layouts, and subtle ambient particle background.
- 🕯️ **Interactive 3D CSS Cake:** Click individual candle flames to blow them out and trigger celebration particle bursts.
- 🎁 **Animated Gift Box:** Tap to open the gift box with lid-bounce physics and reveal the personal letter.
- 🎵 **Built-in Music Synth:** Web Audio API instrumental music generator — no external MP3 hosting required!
- 🎆 **Custom Canvas Engine:** Fireworks, floating balloons, hearts, stars, and confetti explosion physics.
- ✍️ **Dynamic Personalization:** Set recipient details through JS config or directly via URL parameters.
- 📸 **Card Export & Sharing:** Download personalized greeting letter as a PNG image (`html2canvas`) or share via Web Share API.
- 🌓 **Dark & Light Themes:** Toggleable theme mode.
- ⏳ **Celebration Countdown:** Automatic countdown calculator for upcoming birthday dates.
